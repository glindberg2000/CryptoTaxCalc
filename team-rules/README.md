# Team Rules Directory

This directory contains personalized `.cursorrules` files for each team member.

## How to Use

1. **Find your rule file** in this directory (e.g., `fullstackdev.cursorrules`)
2. **Copy the content** to your local `.cursorrules` file
3. **Customize** as needed for your specific setup

## Available Rules

- `fullstackdev.cursorrules` - FullStackDev (Phase 1A completion, merge review)
- `coderdev1.cursorrules` - Coder Dev 1 (FIFO integration)
- `coderdev2.cursorrules` - Coder Dev 2 (Tax logic & fee handlers)
- `additional-fullstackdev.cursorrules` - Additional FullStackDev (Data exploration)
- `template.cursorrules` - Template for new team members

## Rule Structure

Each rule file contains:
- **Role-specific context** and responsibilities
- **Current phase focus** and deadlines
- **Personalized Memory Bank instructions**
- **Team coordination protocols**
- **Technology stack and development guidelines**

## Getting Started

1. Clone the repository: `git clone git@github.com:glindberg2000/CryptoTaxCalc.git`
2. Navigate to team-rules: `cd CryptoTaxApp/team-rules`
3. Copy your rule file: `cp yourname.cursorrules ~/.cursorrules`
4. Customize as needed for your environment

## Updates

Rules will be updated as the project progresses and team assignments change.
Check this directory regularly for updates.
